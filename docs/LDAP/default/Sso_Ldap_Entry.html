		setAttribute (line 103)
	
 
	
	

Set the $value of the specified $attribute. Can also use array syntax $entry['attribute'] = 'value'.


Using this syntax will replace other attributes of the same name. To add a new attribute with that name  use $entry['attribute:'] = 'value'; note the trailing ':' in the attribute name.
